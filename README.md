## What is DP Roast?

DP Roast is a fun, Pakistani-style roasting app for WhatsApp DPs 
and statuses. Select your DP category, set the roast intensity, 
and get a funny culturally-relatable roast — desi humor at its best!

No photo uploads. No personal attacks. Just pure Pakistani Gen-Z fun. 😄

---

## Features

- 🖼️ **17 DP Categories** — Gym Selfie, Nature, Car Pic, Couple, 
  Mirror Selfie, Travel, Religious, No DP, and more
- ✍️ **Caption Roast Mode** — Type your WhatsApp status, get roasted
- 🔥 **3 Intensity Levels** — Halka Phulka 😊 | Thoda Tez 😏 | Khatarnak 🔥
- 🧠 **Personality Quiz** — "Tumhari DP Energy Kya Hai?" (6 questions)
- ⚔️ **Battle Mode** — 2 logon ki DP ka muqabla, winner declare hoga
- 🎵 **Sound Effects** — Web Audio API generated tones (no external files)
- 📸 **Share as Image** — 1080x1080 Canvas export (Instagram story ready)
- 💬 **WhatsApp Share** — Pre-filled roast message
- 🏆 **Hall of Fame** — Leaderboard of most savage roasts (localStorage)
- 🌙 **Dark Mode** — WhatsApp dark theme inspired
- 🎲 **Surprise Me** — Random
